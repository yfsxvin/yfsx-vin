# YFSX & VIN | Open-Source Dual-Token DeFi Ecosystem

**YFSX & VIN** is a fully open-source, community-driven dual-token DeFi ecosystem built on **BNB Smart Chain (BSC)**.

- **YFSX** — Governance & Deflationary Token  
- **VIN** — Lending & Safety Utility Token  

The project follows the principle of **Code is Law · Community is the Future**.  
Fair launch with **no team allocation, no premine, no VC funding**, and no backdoors. All contracts are verified and fully transparent on-chain.

### Token Overview

**YFSX (Governance & Deflationary)**
- Fixed total supply: **19,999** tokens (100% circulating)
- Utilities: On-chain governance, voting, staking rewards, dashboard access
- Transaction fee: 3% (0.3% auto-burn + 2.7% added to liquidity)

**VIN (Lending & Safety Utility)**
- Circulating supply: ≈ **16,670,326** tokens
- Utilities: Non-custodial P2P lending, rate switching, Safety Module (up to 60% shortfall coverage), liquidity incentives
- Transaction fee: 2% (0.3% auto-burn + 1.7% added to liquidity)

The dual-token model creates a self-reinforcing flywheel: fees are burned and liquidity is compounded automatically, supporting long-term sustainability.
