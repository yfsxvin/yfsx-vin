# YFSX-VIN Project Information

YFSX-VIN is a decentralized DeFi ecosystem on BSC with 19,999 YFSX tokens and 16,624,000 VIN tokens. There is no premine, no team allocation, and no VC involvement, making it fully community-owned and deflationary through auto-burn.

The dual-token model consists of YFSX as the governance token and VIN as the ecosystem/rewards token.

YFSX-VIN is actively traded on PancakeSwap V2 and other DEXs, featuring on-chain governance and built on the principle of code is law.
